How to use? Import both jQuery and fancyCountdown .js files then css file, create a div with an id "anyID" then paste the following code between script tags;

$(function() {
$("#anyID").fancyCountdown({
'year':2011,
'month':05,
'day':13,
'hour':13,
'minute':55,
'second':00,
'timeReachedMessage':'Hoorraay!!',
'timeStamps':['Days', 'Hours', 'Minutes', 'Seconds']
});
});
Customization? All parameters can be customized;
First six parameters are
  • "year" -> target year
  • "month" -> target month of that year
  • "day" -> target day of that month
  • "hour" -> target hour of that day
  • "minute" -> target minute of that hour
  • "second" -> target second of that minute
Next two parameters are
  • "timeReachedMessage" -> this is the message appears when the timer ends
  • "timeStamps" -> this is an array to display timestamps in your own language
